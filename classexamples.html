Explanations

This page focuses on manipulating CSS class properties with JavaScript and standard JavaScript events. various classes are altered using several different class types. The following are utilized: getElementsByTagName, getElementById, and getElementsByClassName. Within these, I alter the font colors, font family, alignment, border, and the background color and image.

In order to make each of these events happen, I utilize the standard JavaScript events of onclick, on onmouseover, and ontouch. I could go through and add more events, but thought there were enough gimicks on the page already, and they each work in very similar ways.

The code is my own with the exception of the functions to move the button. This was borrowed from a forum post on webdeveloper.com
