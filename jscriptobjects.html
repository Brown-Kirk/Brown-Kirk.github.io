Variables: Variables are one of the most frequent components of any script, since they are used as placeholders for values the script uses. I have several throughout my script, but a few examples are:

	var i = 1 ;

which declares a variable named "i" and assigns it the number 1. Also, 

	var number = parseInt(document.getElementById('box1').value);

which declares the variable "number" and takes the number from the text box and assigns it. And then there is this one:

	valueListOutput = valueListOutput + "The sum of all the odd numbers between zero and " + i + " is " + valueList[i] + "." + '
';

which takes an already declared variable, and the sentence listed to what is aleady assigned in that variable. Another common variable action is this one:

i++

which takes a numeric variable and increases it by 1. So if "i" equals 5, and it does "i++", then the new value is 6.

Parameters: Parameters are values that are passed into a function. Remember above where we declared the variable "number" and assigned it a value? Well, what we did next was to pass that value into two functions.

	oddSum(number);
oddList(number);

If we follow that into the function, we can see what happens with that variable:

	function oddSum(number1) {
	  var i = 1 ;
	  var result = 0;
	  if (number1 < 1 ) {
	    return 0;
	  }
	  for (i = number1 ; i > 0; i--) {
	    if (i % 2 != 0) {
	      result = result + i;
	    }
	  }
	  resultOutput = "The sum of all the odd numbers between zero and " + number1 + " is " + result + "."
	  document.getElementById('outputdiv').innerHTML = resultOutput;
	}

The "number" variable that was passed into the function becomes the new variable "number1" and is used throughout the function.

Associative Arrays: In JavaScript, associative arrays are not the same as what we normally think of as associative arrays. Rather than being multi-dimensional, they are simply arrays with named indexes rather than numbered. In the example below, rather than an index of 1, the index is "isItOdd". 

	if (i % 2 != 0) {
	  result = result + i;
	  var oddNumber = [];
	  oddNumber["isItOdd"] = "isItOdd";
	} else {
	  var oddNumber = [];
	  oddNumber["isItOdd"] = "even";
	}

In order to retrieve this value, instead of saying "oddnumber[1]", we would say "oddNumber["isItOdd"]"
